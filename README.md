Vị trí 2: Blockchain-based Article Metadata Verification System using Hyperledger Fabric
Số lượng: 02
Ngành học phù hợp: IA, SE, AI
Mô tả công việc:
- Triển khai hệ thống blockchain trên nền tảng Hyperledger.
- Lập trình các hợp đồng thông minh (Chaincode) trên nền tảng Hyperledger Fabric
để băm và lưu trữ metadata bài viết một cách minh bạch, chống giả mạo
- Thiết kế quy trình tự động trích xuất metadata từ URL và kết nối đầu ra của AI
vào hệ thống lưu trữ của Blockchain
- Phát triển giao diện web hoặc API cho phép người dùng nhập URL và nhận kết
quả xác thực.


Yêu cầu:
- Có kiến thức về Lập trình Python (phục vụ AI) và tư duy về hệ thống phân tán
(Blockchain)
- Biết sử dụng Docker/Linux để thiết lập môi trường Hyperledger
- Đã từng hoàn thành hoàn thiện project từ Frontend đến Backend
